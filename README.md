<h3>Installation</h3>
<p>Log into your WordPress backend, on the left menu, click <strong>Plugins</strong> -&gt; <strong>Add New</strong>, then click the button <strong>Upload Plugin</strong>.</p>
<p>2: Select the <strong>wp-performance-test.zip</strong> file and click the button <strong>Install Now</strong>.</p>
<p>3: Click <strong>Activate</strong>.</p>
<p>4. <strong>Enjoy</strong>.</p>
