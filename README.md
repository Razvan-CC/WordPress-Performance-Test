### Installation
 1: Log into your WordPress backend, on the left menu, click **Plugins **-> **Add New**, then click the button **Upload Plugin**.
2: Select the **wp-performance-test.zip**  file and click the button **Install Now**.
3: Click  **Activate**.
<center>**Done**</center>
